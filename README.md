# TechManualRAG

Система мультимодальной обработки технических руководств.
Компактная RAG-система для поиска по инженерной документации (PDF с текстом, схемами, таблицами, формулами).

## Ограничения
- Локальное развертывание требует наличия запущенной инстанции Ollama с установленной мультимодальной моделью (например, LLaVA).
- Для извлечения формул могут потребоваться дополнительные легковесные инструменты типа Marker.
- Требуется tesseract-ocr и poppler-utils для системных зависимостей.

## Запуск
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Инструкция по добавлению новых руководств (План)
1. Разместить PDF в директории `data/`.
2. Система автоматически распарсит PDF (текст, таблицы, схемы).
3. Создаются векторные эмбеддинги (BGE для текста, CLIP для изображений) и сохраняются в локальный Qdrant.
4. Откройте веб-интерфейс для поиска.
