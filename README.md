# TechManualRAG

Мультимодальная RAG-система для поиска по инженерной документации (PDF с текстом, схемами,
таблицами, формулами).

> **Статус:** 🚧 Ранний каркас. Сейчас реализован только Streamlit-интерфейс (загрузка PDF и
> поле запроса); парсинг, индексация и поиск — в разработке (см. «Планируемая архитектура»).
> Это заготовка архитектуры, а не рабочая система.

## Запуск (UI-каркас)
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Планируемая архитектура
- **Парсинг PDF:** текст, изображения (кропы схем), таблицы.
- **Эмбеддинги:** BGE для текста, CLIP для изображений.
- **Векторное хранилище:** локальный Qdrant.
- **Мультимодальная модель:** Ollama + LLaVA (локально).
- **Системные зависимости:** tesseract-ocr, poppler-utils; возможно Marker для извлечения формул.

## План: добавление новых руководств
1. Разместить PDF в директории `data/`.
2. Система распарсит PDF (текст, таблицы, схемы).
3. Создаются эмбеддинги (BGE для текста, CLIP для изображений) и сохраняются в Qdrant.
4. Поиск через веб-интерфейс.
